Navigation bar is used using the 'navbar' class. Inside the navbar div; 'logo' and 'nav-aside' class is used for the logo, and navigation links respectively.


             <div class="navbar">
                 <div class="logo">
                        <h2>Logo
                </div>
                <nav class="nav-aside">
                    <ul class="nav-ul">
                            <li>Home
                            <li>Products
                            <li>About
                    </ul>
                </nav>
            </div>
