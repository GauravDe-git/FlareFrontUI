The toast component can be used using the "wrapper" class. Inside this class use the"toast" class. Insert the "content" class inside.


            <div class="wrapper">
                <div class="toast">
                    <div class="content">
                        <div class="icon">
                            <i class="fa-solid fa-wifi">
                        </div>
                        <div class="details">
                            <span> You are back online! 
                            <p>Internet is connected
                            </p>
                        </div>
                    </div>
                    <i class="fa-solid fa-xmark">
                </div>
            </div>
