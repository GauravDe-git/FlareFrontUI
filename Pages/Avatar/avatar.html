Avatar


            @import url("/Pages/Avatar/avatar.css");
            <link rel="stylesheet" href="/Pages/Avatar/avatar.css">

Avatar with different sizes:

Round avatars have 3 sizes. These are avatar-lg(7rem), avatar-md(5rem) and avatar-sm(3rem).

Square avatars need a avatar-square class to be used.

Put the avatar class inside img tag to use this component.


            < img src="" alt="" class="avatar avatar-lg">
            < img src="" alt="" class="avatar avatar-md">
            < img src="" alt="" class="avatar avatar-sm">
            < img src="" alt="" class="avatar avatar-square">
