Button


            @import url("/Pages/Button/button.css");
            <link rel="stylesheet" href="/Pages/Button/button.css">

Primary Button:

To use the Primary button component; use the class 'btn' followed by the type of button i.e. 'btn-primary', 'btn-danger', etc. inside a button tag.


            <button type="button" class="btn btn-primary"> Primary Color </button>
            <button type="button" class="btn btn-secondary"> Secondary Color </button>
            <button type="button" class="btn btn-success"> Success Color </button>
            <button type="button" class="btn btn-warning"> Warning Color </button>
            <button type="button" class="btn btn-danger"> Danger Color </button>

Link Button:

To use the Link button; add the 'btn-link' class, and add <a> tag inside the button tag to include the link.


            <button type="button" class="btn btn-primary btn-link"> 
                <a href="link"> Link Button </a>    
            </button>
            <button type="button" class="btn btn-secondary btn-link"> 
                <a href="link"> Link Button </a>    
            </button>

Icon Button:

To use the Icon button; add the 'btn-icon' class, and add <img> tag inside the button tag to include the icon.


            <button type="button" class="btn btn-icon btn-primary"> 
                <img src="/Assets/mail.svg"> Icon button 
            </button>
            <button type="button" class="btn btn-icon btn-secondary"> 
                <img src="/Assets/cart.svg"> Icon button 
            </button>

Floating Action Button:

To use the Floating action button; add the 'btn-float' class, and add <img> tag inside the button tag to include the floating action icon.


            <button type="button" class="btn btn-warning btn-float"> 
                <img src="/...">
            </button>
