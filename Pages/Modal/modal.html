Modal component can be used using the class "modal-container" along with the same id. Inside this div, use another class called "modal".


            <button id="open" class="btn btn-primary"> Open Modal 
            <div class="modal-container" id="modal_container">
                <div class="modal">
                    <h1>This is a modal
                    <p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Enim, modi.

                    <button id="close " class="btn btn-primary">Close Modal
                </div>
            </div>
