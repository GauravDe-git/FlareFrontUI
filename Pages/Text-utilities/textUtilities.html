Headings can be used by using the classes "heading1" upto "heading6" for h1 to h6 heading tags.


            <h1 class="heading1">Heading - H1</h1>
            <h2 class="heading2">Heading - H2</h2>
            <h3 class="heading3">Heading - H3</h3>
            <h4 class="heading4">Heading - H4</h4>
            <h5 class="heading5">Heading - H5</h5>
            <h6 class="heading6">Heading - H6</h6>

Small Text:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Reprehenderit rerum quibusdam repudiandae, suscipit minus eligendi.

To use small text, use the class 'text-small' inside a div.


            <div class="text-small">Lorem ipsum dolor sit amet consectetur adipisicing elit. Reprehenderit rerum quibusdam repudiandae, suscipit minus eligendi.</div>
