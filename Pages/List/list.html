Normal List:

    Cities
  • Kolkata
  • Bangalore
  • Chennai

List component can be used with the class "list" in the ul tag."list-title" has to be used in a div tag to showcase the list heading.


            <ul class="list">
                <div class="list-title">Cities</div>
                <li>Kolkata</li>
                <li>Bangalore</li>
                <li>Chennai</li>
            </ul>
