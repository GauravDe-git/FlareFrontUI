Round Image:

Round Images can be used by using the class 'img-round' inside an img tag.


            <img src="/Assets/imgExample.jpg" alt="" class="img-round">

Responsive Image:

Responsive Images can be used by using the class 'img-responsive' inside an img tag.


            <img src="/Assets/imgExample.jpg" alt="" class="img-responsive">
